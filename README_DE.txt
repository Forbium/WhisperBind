Dieses Addon wurde von Forbium und Rogolist geschrieben.
Das Whisper Bind-Addon ist so geschrieben, dass Sie durch Drücken von Tasten einem bestimmten Spieler zuflüstern
können, ohne einen separaten Steckplatz für das Makro zu belegen.

Verwendung.
Um das Flüsterverhalten zu konfigurieren, müssen Sie zu den Add-on-Einstellungen gehen, indem Sie auf die Schaltfläche
„Einstellungen“ klicken. Es öffnet sich ein Fenster mit drei Spalten mit Feldern. In der ersten Spalte schreiben wir,
wie die Schaltfläche heißen soll (um sie leichter zu finden), in der zweiten Spalte, wer sie braucht  und im dritten
Spalte sreiben, welche Nachricht wir senden möchten. Nachdem Sie die Einstellungen abgeschlossen haben, müssen Sie
die Taste drücken Klicken Sie auf „Speichern“ und Sie können das Einstellungsfenster schließen. Beim nächsten Start
Ihres Kontos müssen Sie diese Informationen nun nicht mehr in die Einstellungen eingeben, sie werden automatisch geladen.

Gut geeignet für das Whispercast-Addon, wenn Sie nicht viele verschiedene Nachrichten senden müssen.
Wenn das Fenster über den Bildschirm hinausgeht, müssen Sie den Befehl /run whisperbind:SetPoint("TOPLEFT",600,-200); schreiben.
Dadurch wird das Fenster in die Mitte des Bildschirms gebracht.
