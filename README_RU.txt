Использование.
Для настройки шепота нужно зайти в настройки аддона нажав на кнопку "настройки", она откроет окно где будет три столбца
с полями. В первом столбце пишем, то как мы хотим чтобы называлась кнопка (чтобы легко найти), во втором, кому нужно
отправить сообщение и в третьем, какое сообщение мы хотим отправить. После окончания настройки нужно нажать кнопку
"сохранить" и можно закрывать окно настроек. Теперь в следующий раз когда будет запусщен аккаунт, не нужно будет вводить
эту информацию в настройки, она автоматически загрузится.

Хорошо подходит для аддона висперкаст если не требуется отсылать много различных сообщений.
В случае если окно вышло за рамки экрана, нужно написать команду /run whisperbind:SetPoint("TOPLEFT",600,-200);.
Это перенесет окно в центр экрана.

Использование:
Для настройки шепота нужно открыть окно настроек аддона нажав на кнопку "настройки"("options"), после нажатия откроется
окно в котором будет 4 колонки:
    1.ButtonID  - отображение названия кнопки
    2.Player    - игрок которому будет отправленно сообщение
    3.Message   - текст сообщения
    4.Category  - категория к которой относится сообщение (для удобной фильтрации)
также над колонками есть 4 кнопки:
    1.Save  - сохраняет все значения из полей в окне настроек
    2.New   - добавляет еще одну строку с кнопкой для сообщения
    3.Del   - показывает/скрывает кнопки для удаления строк
    4.Adv   - показывает/скрывает кнопки продвинутых возможностей а именно 5 кнопок:

        1.Stats     - выводит в чат статистику о нажатии текущих привязок (кнопок)
        2.History   - выводит в чат название нажатой кнопки и время нажатия (только последнии 10 нажатий)
        3.Export    - выводит настройки в чат
        4.Import    - (информация в чате)
        5.Auto      - показывает/скрывает настройку автоответа.
Настройка автоответа происходит с использованием одного поля ввода информации (вписывается слово которое будет знаком
для использования ответа) и чекбокса (включает и отключает функцию автоответа(только сообщения у которых отмечена галочка
смогут испоьлзовать автоответ))
На главном окне над кнопкой "настройки" есть кнопка "категории"("Cat:") при нажатии на кнопку переключается фильтр на следующую
категорию. По умолчанию есть 6 категорий (Все, Торговля, Гильдия, Рейд, ПвП, Общение) только у этих категорий есть свои
отличительные цвета, которые будут отображатся на кнопке. Если в настройках в поле для категорий вписать отсутствующую категорию,
то фильтр для нее тоже добавится. При нажатии ПКМ по кнопке "категории" в чат будет выписан список всех доступных категорий.

в сообщениях также можно использовать некоторые переменые: %player%, %myname%, %time%, %zone%, %guild%, %level%, %class%

Хорошо подходит для аддона WhisperCast если не требуется отсылать много различных сообщений.
